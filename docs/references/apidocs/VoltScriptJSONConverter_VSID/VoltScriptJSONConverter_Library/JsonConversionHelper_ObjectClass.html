        %REM
        The code is an abbreviated version of code found in JsonConversionHelper.fromJson(). For a given JSON object (child) gets a converter based on the JSON object's label (elemName) and, if the converter exists, calls its deserialize method to convert the JSON object into mainHolder.

        End If

      • reset

        Public Function reset() As JsonConversionHelper
        Resets the holder object containing the return value. This allows you to re-use the helper.
         
        Code Sample
        %REM
        Uses a helper to deserialize JSON to a Person object, then resets it so it can be re-used by subsequent code. The mainHolder property will be re-initialized, but all converters or labels to ignore will still exist in the helper. So subsequent calls will also ignore "gender" label.
        %END REM

        Dim helper as New JsonConversionHelper()
        Dim jd as Person
        json = |{"firstName":"John","lastName":"Doe","age":42,"gender":"M"}|
        Set jd = helper.ignoreLabel("gender").fromJsonString(json, "Person", "PersonClass")
        Call helper.reset()
