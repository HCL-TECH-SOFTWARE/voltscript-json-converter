Class JsonSetterConverter

      • VoltScriptJSONConverter.VoltScriptJSONConverter.JsonSetterConverter

  • Custom Converter to convert between a JSON value or values and VoltScript datatype or objects, using a specific sub or function in the underlying class. During deserialization, the relevant JSON objects will then be removed from the parent JSON object, to ensure they are not re-processed.

      • withValueParam

        Function withValueParam() As JsonSetterConverter
        Uses the scalar value of the JSON element the converter is associated with.
        The value will be added to the end of the params array, as the nth parameter to pass to the relevant setter.
         
        Code Sample
        %REM
        The following code is from deser-50.txt. It deserializes a JSON object that contains firstName and lastName properties. During the deserialization it calls Laureate.setName(firstName as String, lastName as String). To do this, it uses a JsonSetterConverter tied to the firstName property, so withValueParam picks up that property.
        %END REM
        Dim json as String
        Dim helper as new JsonConversionHelper()
        Dim setterConverter as New JsonSetterConverter()
        Dim json as |{"firstname":"Guglielmo", "surname":"Marconi", "share": 2}|
        Call setterConverter.forSetter("setName")._
            withValueParam()._
            withSiblingParam("surname", "")
        Call helper.withCustomCnverter("firstName", setterConverter)
