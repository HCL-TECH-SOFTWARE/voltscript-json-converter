Public Class AbstractJsonFunctionConverter As AbstractJsonConverter

      • VoltScriptJSONConverter.AbstractJsonFunctionConverter

  • Custom Converter to convert between a JSON value or values and VoltScript datatype or objects, using a specific sub or function in the underlying class. Used as the abstract class for JsonGetterConverter and JsonSetterConverter.

      • withCustomConverter

        Public Function withCustomConverter(labelName As String, converter As AbstractJsonConverter) As Variant
        Passes a custom converter to this AbstractJsonFunctionConverter. The converter will be used to convert the relevant label to a VoltScript object rather than a scalar or scalar array.
        idx Argument Data Type Attributes Comments
        1labelNameStringLabel to look for in the parent JSON object 
        2converterAbstractJsonConverterAbstractJsonConverter to use when deserializing the value 
         
        Code Sample
        %REM
        Run a class's setName(firstName, lastName) function, but convert last name to upper case before passing it in.
        %END REM
        Dim converter as New JsonSetterConverter
        Dim upperCaseConverter as New ConvertToUCaseConverter()
        Call converter.forSetter("setName").withValueParam().withSiblingParam("lastName")
        Call converter.withCustomConverter("lastName", upperCaseConverter)

        Code Sample
        %REM
        Ensures that when extracting the gender element from a JSON object, the target class's setGender method is called with the literal value "M". 
        This is found in the CustomSetterTest sample, a hard-coded test for validation purposes only.
        %END REM
        Dim helper as New JsonConversionHelper()
        Dim genderConverter as New JsonSetterConverter()
        Call genderConverter.forSetter(setGender).withLiteralParam("M")
        Call helper.withCustomConverter("gender", genderConverter)
